# FileOrganizer

Windows 向けのファイル整理アプリです。選択したフォルダ直下にあるファイルだけを対象にして、拡張子ごとに分類し、実行前に移動先をプレビューできます。

## ダウンロード

最新版は GitHub Releases からダウンロードしてください。

- [FileOrganizer-win-v0.2.0.zip](https://github.com/mtyAI/FileOrganizer/releases/download/v0.2.0/FileOrganizer-win-v0.2.0.zip)

以前の安定版も残しています。

- [FileOrganizer-win-v0.1.9.zip](https://github.com/mtyAI/FileOrganizer/releases/download/v0.1.9/FileOrganizer-win-v0.1.9.zip)

zip を解凍して、`FileOrganizer.exe` を起動します。初回起動時に Windows の警告が出る場合があります。

## 使い方

1. `フォルダを選択` で整理したいフォルダを選びます。
2. 必要なら `出力先を選択` で保存先を指定します。未指定の場合は、整理するフォルダ内に分類フォルダを作ります。
3. `整理プレビュー` で、移動されるファイルと移動先を確認します。
4. 整理したくないファイルは `除外` にします。
5. 問題なければ `整理を実行` を押します。

選択したフォルダの中にあるサブフォルダは、基本的に整理対象外です。フォルダ直下に散らばっているファイルだけを整理します。

## 表示モード

- `シンプル`: フォルダ選択、保存先、プレビュー、実行を中心に表示します。
- `詳細`: 分類ルール、重複グループ、ログ、Undo、バックアップをまとめて確認できます。

## 分類カテゴリ

初期状態では、主に次の種類を分類します。

- 動画素材
- 音声素材
- 画像素材
- フォント
- 3D素材
- ゲーム素材
- 圧縮ファイル
- ドキュメント

詳細設定から、使わないカテゴリの無効化や移動先フォルダ名の変更ができます。

## バックアップとUndo

実行時には、選択中フォルダ内に `_file-organizer-backup` が作成されます。

- `Undo`: 直前の実行で移動したファイルを元の場所へ戻します。
- `バックアップを削除`: 選択中フォルダ内のバックアップを削除します。

バックアップを削除すると、そのバックアップからは復元できません。確認してから削除してください。

## 注意

- 大切なフォルダで使う前に、まずコピーしたテスト用フォルダで試してください。
- 実行前に必ず `整理プレビュー` を確認してください。
- ネットワークドライブや外付けドライブでは、移動に時間がかかる場合があります。
- 同名ファイルがある場合は、上書きせず別名で保存します。
