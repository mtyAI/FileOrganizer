# FileOrganizer

Windows向けの素材整理アプリです。選択したフォルダ内のファイルを拡張子ごとに分類し、実行前に移動先をプレビューできます。

## ダウンロード

最新版は GitHub Releases からダウンロードしてください。

- [FileOrganizer-win-v0.1.6.zip](https://github.com/mtyAI/asset-organizer-mvp/releases/download/v0.1.6/FileOrganizer-win-v0.1.6.zip)

zip を展開して、`FileOrganizer.exe` を起動します。

初回起動時に Windows の警告が出る場合があります。

## 基本的な使い方

1. `フォルダを選択` を押します。
2. 整理したい素材フォルダを選びます。
3. アプリがサブフォルダを含めてファイルをスキャンします。
4. `移動前プレビュー` で、移動されるファイルと移動先を確認します。
5. 必要に応じて、移動したくないファイルを `除外する` にします。
6. 問題なければ `実行` を押します。

実行すると、選択フォルダ内にカテゴリ別フォルダが作られ、対象ファイルが移動されます。

## 分類カテゴリ

初期状態では、主に次の種類の素材を分類します。

- 動画
- 音声
- 画像
- フォント
- 3D素材
- ゲーム素材
- 圧縮ファイル
- ドキュメント

左側の `分類カテゴリ` で、使わないカテゴリはオフにできます。

## 除外フォルダ

整理してほしくないフォルダがある場合は、左側の `除外フォルダ` に追加します。

除外は次のどちらでも指定できます。

- フォルダ名: `backup`
- 選択フォルダからの相対パス: `projectA/backup`

除外フォルダに入っているファイルはスキャン対象から外れます。

## 検出候補

スキャン後、アプリは既存フォルダをカテゴリ別に検出します。

`検出候補` からフォルダを選ぶと、除外フォルダとして追加できます。すでに整理済みのフォルダや、触りたくない作業フォルダを外すときに使います。

## バックアップと元に戻す

実行時には、選択フォルダ内に `_file-organizer-backup` が作成されます。

- `元に戻す`: 直前の実行で移動したファイルを元の場所へ戻します。
- `バックアップを削除`: 選択フォルダ内のバックアップを削除します。

バックアップを削除すると、そのバックアップからは復元できません。必要な確認が終わってから削除してください。

## 注意点

- 大切なフォルダで使う前に、まずコピーしたテスト用フォルダで試してください。
- 実行前に必ず `移動前プレビュー` を確認してください。
- ネットワークドライブや外付けドライブでは、移動に時間がかかる場合があります。
- 同名ファイルがある場合は、上書きせず別名で保存されます。
